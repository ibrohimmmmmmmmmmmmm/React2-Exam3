# 🛒 E-Commerce Frontend Project

## 📌 Project Description
This is a modern e-commerce frontend built with React.  
It includes products, brands, categories, authentication, and admin features.

---

## ⚙️ Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- Axios
- React Router DOM
- Formik + Yup
- i18next (for languages)
- Lucide React icons

---

## 🚀 Features
- Product listing (CRUD)
- Brand management (add, edit, delete)
- Category system
- Authentication system
- Responsive UI (mobile + desktop)
- Dark / Light mode
- Multi-language support
- API integration with backend (Swagger)
